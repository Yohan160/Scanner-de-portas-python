# Scanner de Portas em Python

Um scanner de portas TCP desenvolvido em **Python** com interface web utilizando **Flask**.

## 📌 Sobre o projeto

Este projeto permite escanear as portas TCP de um endereço IP e identificar quais estão abertas. O objetivo foi praticar conceitos de programação, redes de computadores e desenvolvimento web com Flask.

## 🚀 Funcionalidades

* Escaneamento de portas TCP
* Interface web simples e intuitiva
* Exibição das portas abertas
* Animação de carregamento durante o escaneamento

## 🛠️ Tecnologias utilizadas

* Python
* Flask
* HTML5
* CSS3
* Socket (biblioteca padrão do Python)

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/Yohan160/Scanner-de-portas-python.git
```

2. Entre na pasta do projeto:

```bash
cd Scanner-de-portas-python
```

3. Instale as dependências:

```bash
pip install -r requirements.txt
```

4. Execute a aplicação:

```bash
python app.py
```

5. Abra o navegador e acesse:

```
http://127.0.0.1:5000
```
